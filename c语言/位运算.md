# 位运算
## 位运算符 ；
* `&` 按位与运算符，两个位都为1时，结果才为1，否则为0。
* `|` 按位或运算符，两个位有一个为1时，结果为1，否则为0。
* `^` 按位异或运算符，两个位不同时为1时，结果为1，否则为0。
* `~` 按位取反运算符，将一个二进制数的每一位取反，即0变1，1变0。
* `<<` 左移运算符，将一个二进制数的各二进位全部左移若干位，由低位补0。
* `>>` 右移运算符，将一个二进制数的各二进位全部右移若干位，由高位补0。
